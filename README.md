# Home Network Security Assessment Using Kali Linux and Nmap

## Overview
This project demonstrates a home network security assessment performed on a Windows host using Kali Linux and Nmap.

## Objectives
- Build a cybersecurity lab using Oracle VirtualBox
- Verify connectivity between Kali Linux and Windows
- Perform port scanning using Nmap
- Identify running services
- Analyze findings and recommend security improvements

## Tools Used
- Kali Linux
- Oracle VirtualBox
- Nmap
- Windows Command Prompt
